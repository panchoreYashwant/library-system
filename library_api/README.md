# Library API (Minimal)

Problem: Build an API for a library system to manage books, members, and borrowing records.

This scaffold includes models (Mongoose), controllers (functional), routes, validation middleware, and tests (Jest + Supertest + mongodb-memory-server).

Uploaded project structure image (as provided): `/mnt/data/e38bf160-a591-4cba-a59c-5cade1e2c89d.png`

## Run
- install deps: `npm install`
- set MONGO_URI in `.env` or tests will use in-memory mongo
- run: `npm start`
- tests: `npm test`
